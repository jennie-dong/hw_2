# 可视化文件说明

1. `delta_hist.png`：展示了高分选项与低分选项之间奖励差值的分布情况，用于衡量奖励模型对偏好选择的一致性。

   图像向右偏的区域多，说明大多数情况下，被选择的回答比被拒绝的回答获得略高的奖励分数，说明模型对人类偏好有一定对齐。

2. `score_hist.png`: 比较了被选中（higher）与被拒绝（lower）选项的奖励得分分布，用于分析奖励模型在不同偏好上的打分差异。

   从图中可见，被选择的回答整体奖励分数更高，表明模型具有良好的区分能力。但其与被拒绝的回答仍有一定重叠，这既可能是因为模型区分能力仍存在局限性，也可能是因为某些数据本身难以区分。

3. `dpo_comparison_results.csv`：DPO 训练后的模型生成结果举例。

4. `worst_cases.csv`: 奖励模型表现效果最差的若干案例。
